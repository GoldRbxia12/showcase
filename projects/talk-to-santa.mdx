This demo shows how to build a latency-sensitive voice agent for product search. It uses the **Realtime API** for turn-taking, and the **Voice Library** for a branded voice.

## Features

- Real-time conversation with low latency
- Branded synthetic voice
- Web-ready and deployable

## Tech

- **Next.js** for the framework
- **Vercel** for deployment
- **WebRTC** for streaming
